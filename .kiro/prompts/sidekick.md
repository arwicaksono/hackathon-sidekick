# Hackathon Sidekick - Main Menu

*Note: This prompt provides a menu interface and does not perform code analysis.*

Welcome! I am your Hackathon Sidekick! I'll help you build a winning hackathon project.

## What would you like to do?

### 1. Evaluate New Idea
Analyze a project idea and get a score prediction.
Command: `@evaluate-idea`

### 2. Initialize Project
Set up optimal Kiro configuration for your project.
Command: `@init-project`
Requires: Completed evaluation

### 3. Check Progress
See current score and get prioritized next steps.
Command: `@update-progress` then `@progress-report`

### 4. Update Documentation
Maintain DEVLOG and README automatically.
Commands: `@update-devlog` and `@update-readme`

### 5. Build Feature
Use custom agents to build features efficiently.
Command: `@build-feature` (if initialized)

### 6. Get Submission Ready
Final checklist and score validation.
Command: `@submission-check`

---

## Quick Start Workflow

**First time:**
1. `@evaluate-idea` - Score your idea
2. `@init-project` - Set up project
3. Start building!

**During development:**
1. `@build-feature` - Build features
2. `@update-devlog` - Document work
3. `@update-progress` - Track score

**Before submission:**
1. `@progress-report` - Final score check
2. `@submission-check` - Verify requirements
3. Submit!
